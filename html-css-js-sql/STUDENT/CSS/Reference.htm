Quick Reference: CSS-1 Properties

 

* Indicates the property can be inherited.

 

 

Font properties

 

Property

Description

Values/units of measurement

font-family *

Declares a specific font or font family

Font name, or generic family name, such as serif or sans-serif

font-size *

Sets the font size, in either absolute or relative value

Absolute size (pt, px, pc), [length] (ex, em), relative or absolute keywords, [percentage]

font-style *

Declares either italic or normal text

normal, italic

font-weight *

Makes a font heavier or lighter

normal, bold, lighter, bolder, 100, 200, 300, 400, 500, 600, 700, 800, 900

font-variant *

Creates small caps

normal, small-caps

 

Text properties

 

text-decoration *

Creates text effects

underline, line-through, overline, blink, none

text-align *

Sets horizontal alignment of text

left, right, center, justify

text-indent *

Sets the first line indentation

[length] or [percentage]

text-transform *

Changes the casing of letters, capitalizes first letter of each word

capitalize, uppercase, lowercase, none

letter-spacing *

Adjusts the kerning of letters

normal, [length]

Word-spacing *

Adjusts the spacing between words

Normal, [length]

line-height *

Adjusts the leading of text

[integer], [length], or [percentage]

Padding

The padding shorthand property

[length], [percentage]

border-color

Sets the color of a border

[color name] or [Hex / RGB value]

border-style

Sets the style of a border

none, dotted, dashed, solid, double, ridge, groove, inset, outset

border-left

border-right

Sets the right border (IE)

[color], [style], [width]

Border

The border shorthand property

all border values

Width

Sets the width of an image or block-level element

[length], [percentage], auto

Height

Sets the height of an image or block-level element

[length], auto

Float

Flows text around an element, usually an image.

left, right, none

Clear

Determines if an element allows floating elements to wrap on either side

left, right, both, none

 

Color and background properties

 

color *

Sets color for an element

[color name] or [Hex / RGB value]

background-color

Sets the background color for an element

[color name] or [Hex / RGB value], transparent

background-image

Sets an image as the background

URL (relative or absolute path), none

background-repeat

Controls how an image is tiled on screen

repeat, repeat-x, repeat-y, no-repeat

background-attachment

Determines if a background image scrolls with page content or is fixed in position

scroll, fixed

background-position

positions background images applied to a particular element

[length], keywords, percentage values


Positioning properties

position

Controls an elements position

absolute, relative, static

left

Used with position, controls the left coordinate of a positioned element

[length], [percentage], auto

top

Used with position, controls the top coordinate of a positioned element

[length], [percentage], auto

clip

Controls which part of an element is visible, and which part is clipped

[shape-with 4 coordinate length values], auto

overflow

Controls the display of an element whose content exceeds the size of its height and width

none, clip, scroll

z-index

Controls the stacking order of overlapping elements

[integer], auto

visibility

Controls the visibility of an element

inherit, visible, hidden

 

Classification properties

 

display

Controls an elements display type

block, inline, list-item, none

white-space

Controls how white space is treated (NN)

normal, nowrap, pre

list-style-type *

Controls the button or character type of a list item

circle, square, decimal, disc, lower-roman, upper-roman, lower-alpha, upper-alpha, none

list-style-image *

Inserts an image to be used as a button for list items

[URL], none

List-style-position *

Determines whether the list label is displayed inside or outside of the list box

Inside, outside
